# Usage

Do OCR processing of an image with fraktur text:

```sh
$ convert Achleitner-Der_Finanzer.pdf Achleitner-Der_Finanzer.jpg
$ tesseract Achleitner-Der_Finanzer-6.jpg Achleitner-Der_Finanzer-6 -l Fraktur
```
