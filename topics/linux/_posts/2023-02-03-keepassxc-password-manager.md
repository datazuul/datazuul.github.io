This post describes how to install the KeePassXC password manager.

# Specification

* Homepage: <https://keepassxc.org/>
* Current Version: v2.7.4

# Installation

## Ubuntu 22.10

```sh
$ sudo apt install keepassxc
...
$ keepassxc --version
KeePassXC 2.6.6
```

# Usage

Database is stored in a file.
